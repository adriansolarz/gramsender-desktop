# GramSender Desktop

Desktop application for Instagram DM automation.

## Download

Download the latest version from [Releases](https://github.com/adriansolarz/gramsender-desktop/releases).

## Features

- Instagram DM automation
- Multi-account support
- Campaign management
- Reply monitoring
- Analytics dashboard

## Installation

1. Download `InstagramOutreachTool_Setup.exe` from Releases
2. Run the installer
3. Follow the setup guide in the app

## Support

Visit [gramsender.com](https://gramsender.com) for documentation and support.
